## 🇬🇧 English Caption

---

🧠 **I built DataMind — a full-stack AI Data Analyst that turns raw data into insights in minutes.**

No more switching between 5 different tools just to understand a dataset.
DataMind is a single, local-first web app that walks you through the entire data science pipeline — from upload to a Colab-ready notebook.

Here is what it can do 👇

**📊 Instant Data Profiling**
Upload a CSV, Excel, JSON, or Parquet file and get a quality score, column health bars, smart alerts for outliers, skewed columns, and high correlations — in seconds.

**🔍 Advanced EDA in one click**
Pearson & Spearman correlation heatmaps with drill-down pair scatter plots. Violin plots, scatter matrices, 3D scatter charts, and pair plots. Full PCA dimensionality reduction with 2D & 3D projections and explained variance charts.

**📈 Time Series Analysis**
Auto-detect datetime columns, visualize rolling statistics, run ADF stationarity tests, decompose trend & seasonality, and get lag autocorrelation suggestions — all with zero code.

**🧹 Intelligent Data Cleaning**
Per-column null imputation (mean, median, mode, forward fill, backward fill, zero, drop, or custom value). IQR-based outlier capping or removal. Built-in contextual help for every single option — so you always know why you are choosing a strategy.

**⚙️ Feature Engineering Studio**
Label, One-Hot, and Ordinal encoding. Standard, MinMax, and Robust scaling. Datetime decomposition (year, month, day, quarter, etc.). Log1p, sqrt, square transforms. Interaction terms, custom mathematical formulas using pandas syntax, and quantile/uniform binning.

**📋 ML Readiness Scorecard**
A dimensional radar chart that scores your data across Completeness, Variance, Encoding, Balance, and Correlation. Know exactly when your data is ready for modeling.

**🐍 Export as a Colab/Jupyter Notebook**
This is my favorite feature. Every cleaning and engineering choice you make in the UI gets converted into clean, well-commented Python code — exported as a .ipynb file that runs on Google Colab, JupyterLab, or VS Code out of the box.

**📄 Self-Contained HTML Report**
One-click export of a full dark-mode HTML report with embedded Plotly charts — share it with anyone, no tools required.

**🤖 AI Assistant (Gemini 1.5 Flash)**
Built-in data science AI that reads your actual dataset and gives you specific, actionable recommendations. Powered by Google Gemini API with zero native dependencies.

---

Tech Stack:
Python · Streamlit · Pandas · NumPy · Plotly · Scikit-learn · SciPy · StatsModels · Jinja2

Everything runs locally — no cloud, no subscription, no data leaves your machine.

I built this because I was tired of repeating the same data cleaning scripts every single project. Now I just upload, click through the pipeline, and download a ready notebook.
